What is CloudLibz ?

CloudLibz is an initative to bring together all the cloud projects at SCoRe
It provides the necessary tools for you to be cloud savvy and makes
managing your projects effortless and easy.We are an all open source organisation
anyone willing to contribute is welcomed at SCoRe
For further information follow our github page

Our Products

Nodecloud Project

NodeCloud is a standard library to get a single API among common cloud service providers (Ex. AWS, GCP, Azure.. ). This will make building products among multiple cloud services and its services easier for the developer.

Gocloud Project

GoCloud is a standard library to get a single API among common cloud service providers (Ex. AWS, GCP, Azure.. ). This will make building products among multiple cloud services and its services easier for the developer.
